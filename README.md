# CRUD de Usuários (React + Express + Prisma + MongoDB)

Projeto fullstack de um CRUD de usuários, com frontend em React e backend em Node.js + Express,
usando Prisma como ORM e MongoDB como banco de dados.

## Tecnologias

### Backend
- Node.js
- Express
- Prisma ORM
- MongoDB

### Frontend
- React
- Axios
- CSS
